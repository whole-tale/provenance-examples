Replication file for “Electoral Backlash against Climate Policy: A Natural Experiment on Retrospective Voting and Local Resistance to Public Policy,” American Journal of Political Science, Leah C. Stokes


DATA
1) replication_file.csv - the analysis dataset
2) newspapers_count.csv - data on the number of newspapers articles in Canada by year with three terms: ``wind energy", ``Ontario" and ``protest". Data was complied through LexisNexis Academic. 
3) precincts0711.csv - matched 2007 and 2011 precincts (polygons)

METADATA
4) codebook.pdf - codebook for replication_file, the analysis dataset

CODE
5) Stokes_AJPS_Replication.R - code to replicate figures and tables
